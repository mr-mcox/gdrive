Installation
=============
Run the following command from the Annaconda Command Prompt

*pip install --upgrade git+https://github.com/mr-mcox/gdrive.git@v0.1.2#egg=gdrive*

Initial setup
=============
Sync the oauth_configs Box folder to your computer
